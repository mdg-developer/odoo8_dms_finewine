SMS Notification

This application provides facility to send SMS notification to the mobile number. For sending SMS, this base application requires an "SMS Gateway"(like- Twilio SMS Gateway, Plivo SMS Gateway, ClickSend SMS Gateway).

Basic Features
  • The module helps to maintain the sent SMS history
  • SMS notification will be sent using SMS gateway.
  • Admin can create globally accessible SMS Templates.
  • Helps to send messages using SMS templates.
  • SMS Templates can be configured on the basis of Sale and invoice status that will be triggered automatically.
  • Multiple SMS templates can be sent at a time.
  • If Customer's mobile number is not defined, then the Customer's phone number will be used for sending SMS.
